# Guide to Contributing

> mOS32

## Rules

**#01** - You must know how C works and have a lot of experience with it.

**#02** - You must know the basics of C/C++ and Assembly (GNU AS).

**#03** - You must follow the Linux Kernel coding style guide.
- [coding-style.html](https://docs.kernel.org/process/coding-style.html)

**#04** - You must know the basics of OS development.
- [OSDev intro](https://wiki.osdev.org/Introduction)
- [Required knowledge](https://wiki.osdev.org/Required_Knowledge)

**#05** - No AI Code: Use of AI is acceptable only by 
- Help debug (read logs).
- Make ideas.
- Search of Documents online.
- Help understand Documents.
